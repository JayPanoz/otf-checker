Open Type Checker

This is a minimum viable web app to test most latin open type features.

Left column is the old syntax (font-feature-settings).
Right column is the new one (font-variant-*).

Please update to the latest version of your browser to get the best results.
If a feature is not supported by your browser, opacity is applied to the text.
